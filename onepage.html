ONE Page Responsive Web


Main Idea Of this project

It is 2018 and everybody has their smartphones, Obviously everybody wants portability, nobody will carry Huge 15 inches Laptop every time. So main purpose of the project is to get started with responsive technology. Responsive design is a way to put together a website so that it automatically scales its content and elements to match the screen size on which it is viewed.

How to get Started?

You can make Responsive Websites/Pages with the help of HTML,CSS and JavaScript, you can start learning that languages from here :
  • W3Schools
  • TutorialsPoint
  • SoloLearn
But if you do want to use above languages than you can use Frameworks like Bootstrap, you will find bootstrap tutorials in above two websites.


Hear is the Example of responsive website.
