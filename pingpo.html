Ping Pong Game

Main Idea Of this project


Main idea of our project is to make 2D ping pong game which is arcade game and mostly we all played that game. You can also come up with some extra things like as time goes ball speed increases or you can put some powers into that which will be little bit complicated but it will fun. Our game will be 2 player game and will be controlled by one keyboard. You can design your own graphics on photoshop or you can use basic elementary object like rectangle for racket and circle for ball.


Resources to learn required technology from basic

technology requirement: C# , basic Unity Engine knowledge, basic understanding of vector algebra
Resources:
  •    https://unity3d.com/learn/tutorials
  •    You can learn from youtube channels like...Brackeys, Blackthornprod.


Minimum functionalities they need to implement

  1.   Should be a 2 player game controlled by one keyboard
  2.   There should be scoreboard to calculate score. Player who score 10 points first will be the winner.
  3.   Appropriate physics will be required for ball to move.
